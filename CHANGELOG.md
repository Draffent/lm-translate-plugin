# Changelog

## [1.0.0] — 2026-06-25

### Added
- 内联翻译：扫描页面英文文本 → 批量百度翻译 → 原文下方显示中文
- 搜索框翻译按钮：中文搜索词自动转英文（词典 + 百度 API）
- FAB 浮动按钮：右下角"翻译此页"，仅搜索/发现页显示
- 侧边栏第5按钮：快捷导航到模型发现页
- 诊断面板：启动时显示连接状态，3秒自动隐藏，悬停保持，手动关闭后页面切换才重新弹出
- 本地 HTTP 代理 (port 18990)：绕过 Electron CORS 限制访问百度翻译 API
- WebSocket 桥接支持（回退方案）
- 翻译还原：点击"还原"删除所有内联翻译
- 路由变化自动清理翻译
- 鼠标悬停保持诊断面板
- 代理连通性静默检查 + FAB 状态指示器
